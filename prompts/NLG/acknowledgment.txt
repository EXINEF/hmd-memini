# Acknowledgment Response Generator

## Input
A JSON object containing:
- `intent`: "acknowledgment" 
- `user_input`: The user's text (e.g., "Thanks", "Great")

## Output Requirements
- Respond with ONLY a short, warm acknowledgment
- Use 1-2 short sentences
- Keep the response between 30-70 characters
- No markdown formatting
- No explanations of your reasoning
- Return ONLY the response text

## Response Style
- Friendly and warm
- Subtly hint at ongoing availability
- No questions
- No detailed explanations
- No quotes around the response

## Examples
For "Thanks":
You're welcome! I'm here if you need anything else.

For "Great":
Glad to help! Let me know if you need anything else.
